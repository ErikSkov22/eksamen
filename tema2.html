02 GRUNDLÆGGENDE WEB

Introduktion

I dette tema fik jeg min første grundlæggende forståelse for, hvordan websites bygges op fra bunden. Jeg arbejdede med HTML og CSS og lærte de basale principper for struktur, layout og responsivt design. Temaet har været fundamentet for resten af mit semester.

Proces

Arbejdsprocessen tog udgangspunkt i et udleveret oplæg med tekst, billeder og designretningslinjer. Jeg arbejdede med semantisk HTML for at skabe en klar og meningsfuld struktur, samt CSS til layout, typografi og responsiv adfærd med fokus på mobile first-princippet. Undervejs anvendte jeg wireframes og mockups som støtte i designprocessen. Der blev løbende arbejdet med strukturering af filer og mapper samt forståelse for sammenhængen mellem design og kode.

Løsning af temaopgave

I denne opgave har jeg udviklet et simpelt website med fokus på struktur, navigation og grundlæggende responsivt design. Sitet er bygget med HTML og CSS og består af en forside samt flere undersider, der er forbundet via en tydelig navigation.

Jeg har arbejdet med semantisk HTML for at skabe en logisk opbygning af indholdet og anvendt CSS til layout, typografi og visuel sammenhæng. Undervejs har jeg haft fokus på en overskuelig filstruktur og brugt browserens developer tools til at teste og rette fejl.

Løsningen viser min grundlæggende forståelse for opbygning af websites og danner fundamentet for mit videre arbejde med frontend og brugeroplevelser.

Læring

Temaet gav mig en grundlæggende forståelse for frontend-udvikling og vigtigheden af god struktur og responsivt design. Jeg lærte at arbejde mere systematisk med kode og fik et solidt teknisk udgangspunkt for de efterfølgende temaer.
